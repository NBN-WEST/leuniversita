# Modulo 04: Beni e Proprietà

## Scopo
Lo studio dei diritti reali, ovvero i diritti sulle cose (res).

## Concetti Chiave
- Concetto di Bene (Materiale/Immateriale, Mobile/Immobile).
- Proprietà (Art. 832 c.c.).
- Modi di acquisto (Originario vs Derivativo).
- Diritti Reali Minori (Godimento vs Garanzia).
- Possesso e Detenzione.
- Usucapione.
