# Modello KPI - Pilot Diritto Privato

Per valutare il successo del programma pilota, monitoreremo quattro dimensioni chiave: Adozione, Impegno, Apprendimento e Soddisfazione.

## 1. Adozione (Adoption)
- **KPI 1.1: Tasso di Attivazione**
  - Formula: `(Studenti attivi / Studenti invitati) * 100`
  - Target: > 80%

## 2. Impegno (Engagement)
- **KPI 2.1: Weekly Active Users (WAU)**
  - Percentuale di studenti che effettuano almeno una sessione di studio a settimana.
  - Target: > 60%
- **KPI 2.2: Tasso di Completamento Piani**
  - Percentuale di attività pianificate completate entro la settimana.
  - Target: > 70%

## 3. Apprendimento (Learning Impact)
- **KPI 3.1: Delta Mastery (Miglioramento Competenze)**
  - Misura la crescita media del punteggio di mastery per topic.
  - Formula: `Mastery Finale - Mastery Iniziale` (su base mensile)
  - Target: +15% medio dopo 4 settimane.
- **KPI 3.2: Riduzione Lacune Critiche**
  - Diminuzione percentuale dei topic con mastery < 40%.

## 4. Soddisfazione (Qualitative)
- **KPI 4.1: Net Promoter Score (NPS)**
  - "Consiglieresti Le Università a un collega?" (Scala 0-10)
  - Target: > 40
- **KPI 4.2: Percezione di Sicurezza**
  - Survey pre/post: "Quanto ti senti sicuro di superare l'esame?"
