# Valore per i Docenti

## 1. Learning Analytics (Vista d'Insieme)
Il docente accede a una dashboard aggregata che mostra la "Mappa delle Competenze" della classe. Questo permette di:
- Identificare argomenti poco chiari a livello collettivo.
- Adattare le lezioni successive basandosi sui dati reali di comprensione.

## 2. Riduzione del Carico di Domande Ripetitive
Il Tutor AI gestisce le domande nozionistiche di base (es. "Differenza tra capacità giuridica e di agire"), liberando il tempo del docente per:
- Ricevimenti di qualità superiore.
- Discussioni su casi complessi e giurisprudenza.

## 3. Supporto alla Creazione di Materiale
I dati sulle lacune frequenti suggeriscono automaticamente dove è necessario integrare le slide o fornire esercizi supplementari.

## 4. Controllo Totale
Il docente ha l'ultima parola sulle fonti. Il sistema ingerisce *solo* il materiale approvato dalla cattedra, garantendo coerenza metodologica.
